## 기간.
 7월29일 - 8월1일

## 개요.
수입 및 지출을 기록하여 항목별로 관리하기 편한 가계부 어플.

## 개발환경.
### HTML & CSS & JavaScript

## 주요기능.
* Local Storage를 이용하여 데이터 저장 (내역 생성 및 삭제)
* sort 함수 및 forEach를 활용한 정렬 기능
* ceil 함수를 이용한 paging처리
* WEB과 MOBILE을 고려한 반응형 레이아웃
